# Reference Solutions

Completed versions of the student TODO files. Use these to:
- Verify your own setup works before teaching
- Help students who get stuck
- Diff against student code during grading

**Do not share this folder with students at the start of class.** Consider keeping it on a separate `solution` branch and merging in just before the workshop ends.
